Feature Description Example
Dynamic Typing Variables can hold any type of data without explicit declaration. let x = 5; x = "Hello";
Event Handling JavaScript can respond to user actions like clicks and key presses. document.getElementById("myButton").addEventListener("click", function() { alert("Button clicked!"); });
DOM Manipulation JavaScript can change the structure and style of web pages dynamically. document.getElementById("myElement").style.color = "red";
